# Pomodoro Telegram Bot

Простой Telegram-бот на Java, который помогает пользователю повысить свою концентрацию и продуктивность с помощью метода Помидора (Pomodoro Technique).

Проект представляет собой Telegram-бота, реализованного на Java, предназначенного для управления Pomodoro-таймерами — рабочими сессиями с чередованием фокусного времени и коротких перерывов. Метод Помидора помогает поддерживать концентрацию, разбивая рабочее время на регулярные интервалы. 
GitHub

Этот бот позволяет пользователю:
- Запускать таймеры Pomodoro;
- Получать уведомления о начале и окончании фокусных сессий и перерывов;
- Управлять временем сессий через простые команды в Telegram;
- Повышать продуктивность с помощью регулярных структурированных промежутков работы.

## Языки и инструменты
- Java
- Gradle
- Telegram Bots API
- Spring Boot
- MongoDB

## Установка
1. Склонируйте репозиторий:
```bash
git clone https://github.com/icestormikk/PomodoroTimerTelegramBot.git
```
2. Перейдите в каталог проекта:
```bash
cd PomodoroTimerTelegramBot
```
3. Откройте проект.
4. Соберите проект с помощью Gradle:
```bash
./gradlew build
```
5. Создайте бота через BotFather в Telegram и получите токен.
6. Добавьте токен в конфигурацию приложения (переменная BOT_TOKEN).
7. Запустите бота
```bash
./gradlew run
```

После запуска бот готов к взаимодействию в Telegram:
1. Отправьте команду /start, чтобы начать работу;
2. Bспользуйте команды для запуска таймеров Pomodoro;
3. Получайте уведомления от бота в соответствии с таймером.
